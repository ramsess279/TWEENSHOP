# TweenShop - E-commerce futuriste

Scaffold initial pour un site e-commerce basé sur React + Vite + TypeScript + Tailwind CSS.

Prérequis : Node.js 18+ et npm ou pnpm.

Démarrage rapide :

```bash
cd tweenshop
npm install
npm run dev
```

Structure minimale :
- src/components : composants UI (Header, Footer, ProductCard..)
- src/pages : Home, Product, Cart
- src/context : logique d'état (CartContext)
- src/data : JSON pour produits, catégories, utilisateurs, réglages
- src/services : services pour interroger les données

Remplace les assets dans `public/assets` par tes logos.
